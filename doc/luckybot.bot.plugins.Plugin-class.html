register_regexp(self, regexp, type, callback, userdata=None)

source code 
Register a new regular expression to match against the incoming data or user message
Parameters:
  • regexp (string) - The regular expression object which will be used for matching
  • type (int) - Where to match against: The Data (REGEXP_RAW) our the user message (REGEXP_MESSAGE)
  • callback (function) - The function to call when the regexp matches
  • userdata - Any object you want to give with the callback
